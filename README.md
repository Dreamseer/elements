## Available scripts

I prefer [yarn](https://yarnpkg.com/), but you can also run these script using [npm](https://www.npmjs.com/get-npm).

### Install and preview elements in Storybook

```bash
yarn && yarn run storybook:start
```

## Run tests

```bash
yarn run test
```

## Run linters

```bash
yarn run lint
```
